# VSafetyWall

This component replaces a single result media view with a full screen "wall"
allowing for opt-in to displaying sensitive media. The component also displays
the specific reasons why a piece of media has been designated sensitive, which
are supplied by the `sensitivity` property of the required `media` prop.
